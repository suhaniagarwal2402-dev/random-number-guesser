# Random Number Guesser 🎮

A simple, interactive Python command-line game where the player tries to guess a randomly generated number within a specific range. It's a great demonstration of basic Python logic, loops, and user input handling.

## Features

* **Dynamic Hints:** Tells the player if their guess is "Too high!" or "Too low!".
* **Score Tracking:** Keeps track of how many attempts it took to guess the correct number.
* **Error Handling:** Gracefully handles invalid inputs (like typing letters instead of numbers).


## Prerequisites

Before running this project, make sure you have Python installed on your system:
* [Download Python](https://www.python.org/downloads/) (Version 3.x recommended)
